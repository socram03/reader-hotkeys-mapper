# ChapterPilot Privacy Policy

Last updated: April 24, 2026

ChapterPilot is a browser extension for manga, manhwa, and web chapter navigation, reading progress recovery, site mapping, and focused reading controls.

## Data Stored by the Extension

ChapterPilot may store:

- Extension settings.
- Keyboard shortcut settings.
- User-created site mappings.
- Supported site activation settings.
- Reading progress and latest-read entries.
- User-created JSON backup/import data.

This data is stored in Chrome extension storage. By default it is stored locally on the user's browser profile. If the user enables the optional sync setting, mappings and settings may be copied to the user's Chrome Sync storage.

## Data Collection and Sharing

ChapterPilot does not:

- Sell user data.
- Use advertising.
- Use analytics.
- Send reading history or mappings to a developer-controlled server.
- Share user data with third parties.

## Website Access

ChapterPilot runs on supported and user-mapped reader pages so it can detect chapter navigation controls, save reading progress, apply keyboard shortcuts, and provide focused reading controls. Page access is used only for extension functionality.

## User Control

Users can clear extension data from the options page, export/import backups, disable custom mappings, disable sync, or remove the extension from the browser.

## Contact

For privacy questions, contact the extension maintainer through the project repository or the support contact listed in the Chrome Web Store listing.
